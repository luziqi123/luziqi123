文章管理

这里应该算是最重要的了，其实也简单，你只需要将你的md文件放到 博客目录/source/_posts 下就OK了，hexo server一下，看是不是多了一篇文章？

需要注意的是，这里只能放md文件，顺便安利一下Typora，Mac下一款超好用的MarkDown编辑器，完全没有了预览窗口，直接用，回车，效果就出来，爽到屙血尿浓啊~
